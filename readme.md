# 🧑‍💻 Portafolio Personal - Fernando Rodriguez

Este proyecto corresponde a la **Práctica Formativa Obligatoria 1 (PFO1)** de la materia Frontend.

Se trata de una landing page de portafolio personal desarrollada utilizando **HTML y CSS**, donde se presentan secciones como información personal, proyectos, habilidades, formulario de contacto y contenido adicional.

El objetivo del trabajo es aplicar buenas prácticas de maquetado, uso de estructura semántica, diseño responsivo y estilización con CSS.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts (Poppins)
- (Opcional) Bootstrap
